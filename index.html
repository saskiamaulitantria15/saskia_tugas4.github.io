Pengertian Polimorfisme

Polimorfisme adalah konsep OOP yang memungkinkan satu method memiliki banyak bentuk.

Method yang sama dapat melakukan aksi berbeda tergantung objeknya.

  • Mengurangi duplikasi kode
  • Meningkatkan fleksibilitas
  • Mudah dikembangkan

Polimorfisme dengan Inheritance

Polimorfisme memungkinkan satu variabel dengan tipe parent menyimpan objek child.

Manfaat Polimorfisme

  • Fleksibilitas
  • Extensibility
  • Code Reusability
  • Maintainability

Operator is dan as

Digunakan untuk pengecekan dan casting tipe data.

Pengertian Abstraksi

Abstraksi adalah proses menyembunyikan detail kompleks dan hanya menampilkan bagian penting.

Abstract Class

Abstract class tidak bisa dibuat objek.

Abstract Method

Method tanpa isi yang wajib diimplementasikan.

Interface di Dart

Menggunakan implements.

Perbedaan Extends dan Implements

  • extends mewarisi isi
  • implements struktur

Static Member & Method

Diakses tanpa objek.

Kesimpulan

Dengan konsep OOP, program menjadi lebih terstruktur dan mudah dikembangkan.
