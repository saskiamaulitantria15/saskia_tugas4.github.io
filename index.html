Pengertian Polimorfisme

Polimorfisme adalah konsep OOP yang memungkinkan satu method memiliki banyak bentuk.

Method yang sama dapat melakukan aksi berbeda tergantung objeknya.

  • Mengurangi duplikasi kode
  • Meningkatkan fleksibilitas
  • Mudah dikembangkan

Polimorfisme dengan Inheritance

Polimorfisme memungkinkan satu variabel dengan tipe parent menyimpan objek child, sehingga method yang sama dapat menghasilkan output berbeda sesuai objeknya. 

// source code sesuai dartpad

Manfaat Polimorfisme

  • Fleksibilitas: Program dapat bekerja dengan berbagai tipe objek tanpa mengubah kode utama.
  • Extensibility: Mudah menambahkan class baru tanpa mengubah struktur program yang sudah ada.
  • Code Reusability: Method yang sama dapat digunakan kembali pada banyak class.
  • Maintainability: Kode lebih mudah diperbaiki dan dikembangkan.

Dengan adanya polimorfisme, program menjadi lebih efisien, rapi, dan mudah dikembangkan di masa depan.

Pengertian Abstraksi

Abstraksi adalah proses menyembunyikan detail kompleks dan hanya menampilkan bagian penting.

  • Menyederhanakan program
  • Fokus pada fungsi utama
  • Meningkatkan keamanan
